# vite-react-typescript-starter

> vite-react 脚手架, 支持 typescript,less, react-router-dom, vite 环境变量, husky 提交验证

## 开始

```shell
yarn
```

## 调试

```shell
yarn dev
```

## 构建

> [vite 环境变量配置](https://cn.vitejs.dev/guide/env-and-mode.html)
> .env.development 与.env.production 环境变量配置

```shell
yarn build
```

## 提交 commit

```shell
yarn commit
```
